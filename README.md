# Wikipedia Search Application

A simple web application that allows users to search Wikipedia articles using the Wikipedia API. Built with HTML, CSS, and JavaScript.

## Features
- Search Wikipedia articles instantly
- Fetch results using Fetch API
- Shows loading spinner while fetching
- Displays title, URL, and description dynamically
- Clean and responsive interface

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Wikipedia Search API
